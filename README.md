# Paddock Club
Formula 1 Paddock Club: the new companion app for F1 fans with a lot of features

# Features

- Calendar
- Next Race
- Week-end Sessions (Practice, Race, Sprint, Qualifying) 
- Today Event
- Qualifying Results
- Sprint Results
- Race Results
